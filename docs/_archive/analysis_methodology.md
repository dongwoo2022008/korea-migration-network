## 1. 분석 기간 설정

본 연구는 분석 목적에 따라 세 개의 기간을 구분하여 운용한다.

| 트랙 | 기간 | 관측치 | 목적 |
| :--- | :---: | :---: | :--- |
| **기술·네트워크 분석** | 2008~2025 (18년) | 4,122 | RQ1 — 네트워크 구조 변화 기술 |
| **패널 계량모형 (주모형)** | 2009~2024 (16년) | 3,616 | RQ2·RQ3·RQ4·RQ6 — 인과 추정 |
| **ML 확장 트랙** | 2017~2024 (8년) | 1,817 | RQ5 — 흡인력 결정요인 탐색 |

**기간 설정 근거**

- **2008년**: 시차 변수($t-1$) 생성 원천으로만 활용. `childcare_pk`(보육시설수) 미집계로 주모형 분석 기간에서 제외.
- **2025년**: 네트워크 지표 및 인구·이동 변수는 완전 수집되었으나, 고용률·실업률·자연증가율 등 통제변수 16개가 공표 주기상 미집계 상태(2026년 하반기 공표 예정). 체계적 결측(Systematic Missing)으로 인한 추정 편의 방지를 위해 계량모형에서 제외하고 네트워크 기술 분석에만 포함.
- **2017년**: 사업체수(`biz_count`), 고용률(`employ_rate`) 등 경제·산업 변수의 시군구 단위 데이터가 완전 확보되는 시점. ML 확장 트랙의 시작점으로 설정.

### 트랙 A — 기술·네트워크 분석 (2008~2025)

**대응 연구질문**: RQ1 — 인구이동 네트워크는 2006~2025년 동안 어떻게 변화하였는가?

#### A-1. 기술통계 분석

- 229개 시군구의 순이동률, 전입률, 전출률 시계열 추이 기술
- 수도권·비수도권, 광역시·중소도시·군 단위 집단별 비교
- 코로나 전후(2019년 이전 / 2020~2022 / 2023 이후) 구조 변화 기술

#### A-2. 인구이동 네트워크 분석

- **분석 단위**: 연도별 229 × 229 OD(Origin-Destination) 행렬
- **산출 지표**:

| 지표 | 의미 | 활용 |
| :--- | :--- | :--- |
| In-Degree Centrality | 유입 연결 수 | 흡인 허브 식별 |
| Out-Degree Centrality | 유출 연결 수 | 방출 허브 식별 |
| In-Strength | 유입 이동량 합계 | 절대적 흡인력 |
| PageRank | 이동망 내 영향력 | 간접 흡인력 |
| Betweenness Centrality | 이동 경로 매개 역할 | 중간 거점 식별 |
| Closeness Centrality | 전체 네트워크 접근성 | 네트워크 위치 |

- **커뮤니티 탐지**: Louvain Algorithm 적용, 연도별 모듈성(Modularity) 산출
- **네트워크 밀도(Density)** 및 **허브 집중도(Gini of In-Strength)** 연도별 추이

#### A-3. 시각화

- 연도별 Choropleth Map (순이동률, PageRank)
- 주요 연도(2008, 2013, 2019, 2024) Network Graph
- 중심성 지표 시계열 Heatmap (상위 30개 시군구)
- Sankey Diagram (주요 이동 흐름, 5개 연도 비교)

#### B-2. 추정 절차 (단계적 모형 구축)

```
Step 1. Pooled OLS
        → 기저 계수 방향 확인, 이분산 검정(Breusch-Pagan)

Step 2. 고정효과 (FE, Within Estimator)
        → 지역 고정효과 + 연도 고정효과 (Two-Way FE)
        → Hausman 검정으로 FE vs RE 선택

Step 3. 공간 자기상관 진단
        → 시군구 Shapefile 기반 공간 가중치 행렬(W) 구성 (Queen Contiguity)
        → Moran's I 검정 (잔차 공간 자기상관 여부)

Step 4. 공간 패널 모형 (SDM, Spatial Durbin Model)
        → 공간 자기상관 확인 시 적용
        → SAR/SEM 제한 검정으로 최적 모형 선택

Step 5. Spatial System-GMM (주모형)
        → 내생성 통제: 네트워크 지표의 2기 이상 시차를 도구변수로 활용
        → Arellano-Bond AR(2) 검정, Sargan/Hansen 과대식별 검정
        → 정방향(t-1→t) 및 역방향(t→t+1) 추정으로 자기강화 루프 검증
```

#### B-3. 이질성 분석 (강건성 검증)

| 분석 | 방법 | 목적 |
| :--- | :--- | :--- |
| 수도권 vs 비수도권 | 더미 상호작용항 또는 하위 표본 분리 추정 | 공간적 이질성 확인 |
| 코로나 전후 | 2020년 기준 구조 변화 검정 (Chow Test) | 시간적 이질성 확인 |
| 도시 규모별 | 특광역시·시·군 분리 추정 | 도시 유형별 이질성 |
| 시도 단위 집계 | 17개 시도 패널로 재추정 | 집계 수준 강건성 |

#### B-4. PVAR 보충 분석 (RQ6 — 결정요인의 시간적 변화)

- **Panel VAR**: 순이동률과 핵심 독립변수(재정자립도, 청년비율, PageRank 등) 간 동태적 상호인과 관계 분석
- **충격반응함수(IRF)**: 각 변수 충격에 대한 순이동률의 반응 경로 추정
- **분산분해(FEVD)**: 순이동률 변동의 요인별 기여도 산출

| 분석 | 출력물 | 목적 |

#### C-5. 지역 흡인력 지수(RAI) 개발

SHAP 값을 활용하여 각 지역의 **지역 흡인력 지수(Regional Attractiveness Index, RAI)**를 산출한다.

```
RAI_i = SHAP_기반_예측값 (표준화)
      = Σ(SHAP_j,i) for j in {경제, 인구, 교육, 의료, SOC, 인프라 변수}
```

- 영역별(경제·인프라·인구·서비스) 부분 RAI 산출
- 연도별 RAI 변화 추이 Choropleth Map 시각화
- RAI 상위·하위 지역의 특성 프로파일링

## 3. 분석 단계별 착수 순서

```
[1단계] 전처리 및 파생변수 생성
        ├─ 시차 변수 생성 (L.pagerank, L.betweenness 등)
        ├─ 변화량 변수 생성 (Δyouth_ratio, Δfiscal_indep 등)
        ├─ 결측 보간 (extinction_risk 등 2008~2010 구간)
        └─ 스케일링 (규모 변수 로그 변환, 비율 변수 z-표준화)

[2단계] 기술통계 및 네트워크 분석 (트랙 A, 2008~2025)
        ├─ 연도별 중심성 지표 시계열 기술
        ├─ 커뮤니티 탐지 및 모듈성 변화
        └─ Choropleth Map, Network Graph 시각화

[3단계] 공간 가중치 행렬(W) 구성 및 Moran's I 진단

[4단계] 패널 계량모형 단계적 추정 (트랙 B, 2009~2024)
        ├─ Pooled OLS → FE → RE → Hausman
        ├─ SDM (공간 자기상관 확인 시)
        └─ Spatial System-GMM (주모형)

[5단계] PVAR 보충 분석 (RQ6)

[6단계] ML 모형 학습 및 SHAP 분석 (트랙 C, 2017~2024)

[7단계] 지역 흡인력 지수(RAI) 산출 및 시각화

[8단계] 이질성 분석 및 강건성 검증

[9단계] 논문 작성
```

## 5. 논문 방법론 기술 요약 (초안)

> 본 연구는 2008~2025년 전국 229개 시군구 패널데이터를 구축하여 분석을 수행한다. 분석 목적에 따라 세 개의 기간을 구분하여 운용한다. 첫째, 인구이동 네트워크의 구조적 변화를 기술하는 탐색적 분석은 네트워크 지표가 완전히 확보된 2008~2025년 전기간을 활용한다. 둘째, 인구이동 결정요인의 인과적 추정을 목적으로 하는 패널 계량모형은 핵심 통제변수의 커버리지가 안정화되는 2009년부터 2024년까지 16개 연도를 분석 기간으로 설정하며, 2008년 데이터는 1기 시차 변수 생성에 활용한다. 2025년은 고용률, 자연증가율 등 주요 통제변수가 공표 주기상 미집계 상태로 체계적 결측에 의한 추정 편의를 방지하기 위해 계량모형에서 제외한다. 셋째, 머신러닝 기반 확장 분석은 사업체수, 고용률 등 경제·산업 변수의 시군구 단위 데이터가 완전히 확보되는 2017~2024년을 분석 기간으로 설정하여 더 풍부한 변수셋으로 지역 흡인력 결정요인을 탐색한다.
