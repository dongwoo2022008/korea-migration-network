## 4. 분석 가능성 종합 판단

### 현재 데이터로 즉시 분석 가능한 항목 ✅

| 연구질문 | 분석 항목 | 가능 여부 |
|----------|-----------|-----------|
| **RQ1** 네트워크 구조 변화 | 중심성 지표 시계열, Louvain, HHI, Freeman | ✅ 완료 |
| **RQ2** 허브 지역 특성 | PageRank 상위 지역 분석, SDM | ✅ 완료 |
| **RQ3** 연령별 네트워크 | 연령별 OD 네트워크 분석 | ❌ OD 원자료 없음 |
| **RQ4** 청년층 단계적 이동 | 20대 이동 경로 추적 | ❌ OD 원자료 없음 |
| **RQ5** 흡인력 결정요인 | ML + SHAP + RAI | ✅ 완료 (종사자수·GRDP 제외) |
| **RQ6** 결정요인 시간 변화 | 기간분할 FE·SHAP | ✅ 완료 (탐색적) |

### 핵심 데이터 갭 요약

> **가장 시급한 갭은 연령별·성별 OD 행렬 원자료이다.** 현재 Track A는 시군구별 집계 이동 지표(in_rate, out_rate, net_rate 등)만 보유하고 있어 **RQ3(연령별 네트워크)과 RQ4(청년층 이동 경로)를 직접 분석할 수 없다.** 통계청 MDIS에서 연령별·성별 OD 행렬을 수집하면 두 연구질문을 모두 해결할 수 있다.

> **종사자수(worker_count)와 GRDP(grdp_pc)의 결측률이 높아** 현재 Track C ML 모형에서 제외되어 있다. KOSIS에서 전 기간 데이터를 보완하면 모형 설명력을 높일 수 있다.
