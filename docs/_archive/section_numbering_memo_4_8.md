## 1. §4.8 절번호 변경 이력

RAI 내적 타당성 검증 소절이 `4.8.3-b`라는 비표준 번호로 삽입되어 있었으며, 이를 정규 번호 체계로 재정비하였다.

| 변경 전 | 변경 후 | 내용 |
|---------|---------|------|
| `### 4.8.1 RAI 지수 산출 결과` | `### 4.8.1. RAI 지수 산출 결과` | 마침표 추가 |
| `### 4.8.2 RAI 공간 분포 (Figure 4-17)` | `### 4.8.2. RAI 공간 분포 (Figure 4-17)` | 마침표 추가 |
| `### 4.8.3 RAI-순이동률 수렴 타당도 (Figure 4-18)` | `### 4.8.3. RAI-순이동률 수렴 타당도 (Figure 4-18)` | 마침표 추가 |
| `### 4.8.3-b RAI 내적 타당성 검증 (Cronbach's α · PCA)` | `### 4.8.4. RAI 내적 타당성 검증 (Cronbach's α · PCA)` | 비표준 `-b` 접미사 → 정규 번호 `4.8.4.` |
| `### 4.8.4 하위지수 구성 비교 (Figure 4-19)` | `### 4.8.5. 하위지수 구성 비교 (Figure 4-19)` | 번호 +1 밀기 + 마침표 |
| `### 4.8.5 소결` | `### 4.8.6. 소결` | 번호 +1 밀기 + 마침표 |

**변경 사유**: `4.8.3-b`는 Markdown 절번호 표준에 어긋나며, 자동 목차 생성 도구(Pandoc, LaTeX 등)에서 인식 오류를 유발할 수 있다. 정규 번호 `4.8.4.`로 교체하여 §4.8 소절 구조를 6단계(4.8.1~4.8.6)로 확정하였다.
